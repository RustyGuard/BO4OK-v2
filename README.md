# BO4OK-v2
Стратегия в реальном времени созданная на pygame

## Требования
Версия python >= 3.10

В данный момент поддерживается только windows

## Запуск проекта

Создать и активировать venv

Установить зависимости
```shell
pip install -r requirements.txt
```
Запустить игру
```shell
python src/main.py
```
